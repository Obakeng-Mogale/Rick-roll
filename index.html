Given an integer array nums and an integer k, implement Rick's Algorithm which finds the maximum sum of any contiguous subarray of length exactly k.

Your solution must run in O(n) time. A naive O(n·k) solution will fail the performance test cases.

Constraints:

• 1 ≤ k ≤ nums.length ≤ 10⁵
• -10⁴ ≤ nums[i] ≤ 10⁴

Example 1
Input: nums = [2,1,5,1,3,2], k = 3
Output: 9
Explanation: [5,1,3] has the maximum sum = 9
Example 2
Input: nums = [-1,4,2,1], k = 2
Output: 6
